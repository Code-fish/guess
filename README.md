## Installation
Change to app directory, use `virtualenv` create and activate virtual enviroment.  
Then use `pip` to install requirements：  
```
pip install -r requirements.txt
```
Run:  
```
set FLASK_APP=app.py    
flask run
```

Go to http://127.0.0.1:5000/
